# One-Dream

## 个人平时使用到的工具集

- 常用正则表达式(regular expression)
##### 手机号码(不含区号)
##### 中国身份证
##### 邮箱（不允许有中文)
##### 邮箱（允许有中文)
##### 邮编（中国）
##### 至少8个字符，至少1个大写字母，1个小写字母和1个数字,不能包含特殊字符（非数字字母)
##### 至少8个字符，至少1个字母，1个数字和1个特殊字符（如果还需要匹配其他字符请自行添加)
##### 至少8个字符，至少1个大写字母，1个小写字母，1个数字和1个特殊字符（如果还需要匹配其他字符请自行添加）
##### 输入值只能为大于0或者正数，两位小数，其他符号不可输入
